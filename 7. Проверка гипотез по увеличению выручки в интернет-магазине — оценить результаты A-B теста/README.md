# Описание проекта

Проведена приоритизация гипотез по фреймворкам ICE и RICE. Затем был проведен анализ результатов A/B-теста, построены графики кумулятивной выручки, среднего чека, конверсии по группам, а затем посчитал статистическую значимость различий конверсий и средних чеков по сырым и очищенным данным. На основании анализа мной было принято решение о нецелесообразности дальнейшего проведения теста.
